# 📸 PageSnap — 落地页 A/B 测试 & 转化率优化器

> **一句话**: 给你的落地页拍个"X光片"，AI 告诉你哪里在漏钱，自动生成优化版本。

---

## 📊 立体评估

### 1. 市场维度 ⭐⭐⭐⭐⭐ (5/5)

| 指标 | 数据 |
|------|------|
| **TAM** | CRO(转化率优化)市场 $12B+ (2026) |
| **SAM** | 独立开发者 + 微SaaS + 电商 ~3M |
| **SOM** | 第一年 2K-15K 用户 |
| **竞争格局** | Optimizely(企业级$50K+)、VWO(中大型)、Google Optimize(已停运！) |
| **市场缺口** | Google Optimize 停运后，没有免费/轻量的 A/B 测试工具 |

**关键信号:**
- Google Optimize 2023 年停运 → 数百万用户需要替代品
- 独立开发者做 Landing Page 全靠"感觉"，0 数据驱动
- CRO 是 ROI 最高的投资：1% 转化提升 = 巨大的收入增长
- "我的 landing page 转化率很低"在 IndieHackers 上月均 100+ 帖

### 2. 技术维度 ⭐⭐⭐ (3/5)

| 维度 | 评估 |
|------|------|
| **复杂度** | 中高 — 浏览器自动化 + 热力图 + A/B 框架 |
| **核心依赖** | Playwright(截图)、注入 JS(热力图/AB)、AI 分析 |
| **数据存储** | SQLite(本地) 或 Supabase(云端) |
| **AI 层** | 页面分析 + 文案优化建议 + 自动生成变体 |
| **估时** | 3周 MVP，6周完整版 |
| **技术风险** | 🟡 中 — 跨浏览器兼容、JS 注入稳定性 |
